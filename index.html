Software pro vzdálenou podporu

Rychlá pomoc

Po stažení spusťte a předejte IT podpoře zobrazené ID číslo a heslo.

Tato verze se neinstaluje. Jakmile program ukončíte, ztratí vzdálená IT podpora přistup na váš počítač.

Trvalá instalace

Po stažení nainstalujte a předejte IT podpoře zobrazené ID číslo a heslo.

Tato verze na vašem počítači zůstane nainstalována natrvalo. Bude se automaticky spouštět při každém startu systému.
